# Protótipo de WebGIS 🗺️

**Disciplina:** Engenharia de Software II 
**Projeto:** Protótipo de WebGIS  

## 📌 Sobre o Projeto

O **Protótipo de WebGIS** é uma aplicação de mapeamento web desenvolvida para facilitar a vida dos estudantes universitários. O objetivo principal da plataforma é permitir que os alunos visualizem o campus de sua faculdade no mapa e encontrem facilmente pontos de alimentação (restaurantes, lanchonetes, cafés, etc.) localizados nas proximidades.

## 🎯 Público-Alvo e Contexto

A transição e o dia a dia na universidade podem ser corridos. Muitas vezes, os alunos não conhecem as opções de alimentação ao redor do campus. Este projeto resolve esse problema fornecendo uma interface interativa e geolocalizada, focada exclusivamente nas necessidades gastronômicas no entorno do ambiente universitário.

## ✨ Funcionalidades

O mapa interativo conta com as seguintes ferramentas de navegação e usabilidade:

* **Navegação Fluida (Pan):** Permite arrastar e explorar diferentes áreas do mapa livremente.
* **Controle de Zoom:** Aproximação e distanciamento para visualização em macro ou micro escala da região.
* **Camadas de Mapa (Map Types):** Alternância entre diferentes estilos de visualização de mapa (ex: mapa de ruas padrão, visão de satélite/GPS).
* **Geolocalização Integrada:** Captura a localização atual do usuário em tempo real para centralizar o mapa e mostrar os pontos de alimentação mais próximos a ele.
* **Mapeamento de Pontos de Alimentação:** Marcações no mapa indicando os estabelecimentos disponíveis ao redor do campus universitário.

## 🛠️ Como Executar o Projeto Localmente

Certifique-se de ter o [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado na sua máquina.

1. Clone este repositório:
   ```bash
   git clone [https://github.com/rayssamell/webgis_app.git](https://github.com/rayssamell/webgis_app.git)
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd WEBGIS_APP
   ```
3. Instale as dependências do Flutter:
   ```bash
   flutter pub get
   ```
4. Execute a aplicação (para rodar no navegador, já que é um WebGIS):
   ```bash
   flutter run -d chrome
   ```
